Welcome!

Click through the map layers to highlight areas and discover statistics and data about buildings in Canada.
  • Use the previous extent button to move back to the view provided by the previous selection. Note: This does not move through the previous data of the ODB. It will not reselect a previously selected building.
  • Select buildings to receive updated statistics for the updated selection. Using the previous extent button after multiple building selections will still select the extent of the previous dissemination block as the extent is not changed between the odb layers and the dissemination block layers.
  • The application does not zoom into the view of the buiding.
  • You can use the Back previous level button to toggle the selection level up to one level per click. This is likely to give unexpected result for a much larger geography than is accommodated by the current extent/view of the map. Manually zoom out enough to improve the selection accuracy.
  • Statistics and Data will be updated correctly for the full selection, even if the highlighted area is unexpected.
  • Click back to default view button for area (eg. Zoom to Halifax, Nova Scotia) to restore all default settings.
